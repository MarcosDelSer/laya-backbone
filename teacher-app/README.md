# LAYA Teacher App

React Native 0.78 app for educators (care logging, activities, notifications).

## Quick start

- **iOS:** `npx react-native run-ios`
- **Android:** See [../docs/ANDROID_APP_SETUP.md](../docs/ANDROID_APP_SETUP.md) for SDK, emulator, Gradle/Metro fixes, and run steps.  
  Then: `npx react-native run-android` (with emulator running and optionally `npm start` for Metro).

## Scripts

- `npm start` — Metro bundler
- `npm run android` — Run on Android (requires env + emulator; see docs)
- `./run-android-emulator.sh` — Install debug APK and launch (emulator must be running)
